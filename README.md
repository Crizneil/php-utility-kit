# PHP Developer Utility Kit

A collection of lightweight PHP utility scripts and helper functions designed to streamline common development tasks.

## Features
- **Helper Functions**: Modular functions for string manipulation and data formatting.
- **Config Loader**: A simple loader for environment-specific configuration files.
- **Extensible**: Designed to be cloned and adapted for small to medium projects.
